Signs + Types of Media

"Meaningful Media"

Text, image, audio, video + interface as indices
(meaning through indication)

Text, image, audio, video + interface as icons
(meaning through likeness or analogy)

Text, image, audio, video + interface as symbols
(meaning through convention or rule)

Rhetoric + Types of Media

"Persuasive Media"

Text, image, audio, video + interface as ethos
(persuasion through credibility or authority)

Text, image, audio, video + interface as pathos
(persuasion through feeling or emotion)

Text, image, audio, video + interface as logos
(persuasion through logic or reason)

Text, image, audio, video + interface as kairos
(persuasion through time or timing)

Text, image, audio, video + interface as procedure
(persuasion through process)

Examples from 2024-25?

A credible and convincing interface

Text that lacks authority

An image that elicits emotion

Unemotional audio

A logical video

An illogical interface

Poorly timed text

A timely image

A credible, emotional X

A logical, timely Y

Games, as a format, tend to combine all the above through attention to procedure or "procedural rhetoric."
